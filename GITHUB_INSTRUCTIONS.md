# How to Push to GitHub from Replit

1.  **Open the Git Pane:** Look for the Git icon (it looks like a branch) in the left-hand sidebar of your Replit editor.
2.  **Connect GitHub:** If you haven't already, you'll see an option to "Connect to GitHub". Follow the prompts to authorize Replit.
3.  **Create/Link Repository:** 
    *   If the repo doesn't exist, select "Create a new repository on GitHub".
    *   If it exists, you can link it.
4.  **Stage and Commit:** Make sure all your changes are typed into the "Commit message" box and click **Commit & Push**.
5.  **Push:** If you've already committed, look for the up-arrow icon or the "Push" button in the Git pane to send your code to GitHub.

Your code is now being managed by GitHub!
